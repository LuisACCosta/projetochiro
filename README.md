# Projeto Chiro

## Como testar

Para executar os testes locais (Unit Tests) que não requerem um dispositivo Android ou emulador, use o comando:

```bash
./gradlew test
```

Para executar os testes instrumentados (Instrumented Tests) que requerem um emulador ou dispositivo Android conectado, use o comando:

```bash
./gradlew connectedAndroidTest
```
